プライバシーポリシー

KOWA Fortune（以下「本アプリ」）は、利用者の個人情報を尊重し、適切に取り扱います。

収集する情報

本アプリは、占いサービス提供のために必要な情報を取得する場合があります。

情報の利用目的

取得した情報は、以下の目的で利用します。

- サービス提供
- アプリの改善
- お問い合わせ対応
- 不正利用防止

第三者提供

法令に基づく場合を除き、利用者の同意なく第三者へ個人情報を提供しません。

広告について

本アプリは広告サービスを利用する場合があります。

アクセス解析

本アプリは利用状況分析のためアクセス解析ツールを利用する場合があります。

免責事項

本アプリの利用により生じた損害について、当方は責任を負いかねます。
